# [04.10] 게시글 매칭 AI Spark 분산처리

### matching_ai
2. 유사도 계산 개선

텍스트 유사도 함수:

정확한 일치 및 포함 관계 검사 추가
자카드 유사도 계산 추가
중요 키워드에 더 높은 가중치 부여


카테고리 유사도 함수:

ID 비교 기능 추가
포함 관계 검사 개선
최소 유사도 보장 로직 추가


필드별 유사도 계산 개선:

물품명: 완전 일치/포함 관계 시 높은 점수 부여
색상: 일치/공통 키워드 감지 시 높은 점수 부여
내용: 공통 키워드 비율 기반 점수 계산
위치 정보 추가: 위치 정보로 유사도 향상

- 사용자가 습득물 게시글을 올리면 분실물 DB 전체와 비교를 해 유사한 게시글 뽑아내도록
